# Robotics School System

## Project Description
Web system developed in the most recent version of Laravel for a robotics institution. The platform handles user management across three roles (students, teachers, and admin). The course administration includes specific robotics kits. Students can be assigned to different groups with varying levels of mastery, which can be monitored by the teachers.
